# CM3D2.ConsoleCodePage.Plugin

コンソールの文字化けを直すUnityInjectorプラグイン 

## 説明

その２>>779以降で発表されている[asm256](https://github.com/asm256)氏による「[CCM3D2.ConsoleCodePage.Plugin.cs](https://gist.github.com/asm256/9bfb88336a1433e2328a)」をダウンロード、コンパイルします

以下のコマンドで、ダウンロードおよびコンパイルを行います

```
compile.bat
```

## なんでこのプラグインだけ？

「文字化けしているけど正常なの？」という疑問を減らすために、また主に当リポジトリ管理者の動作テストが楽になるため、入れてあります。
